# TPU Boundary Changes Analysis (2001-2021)

## Overview

This report analyzes changes in Tertiary Planning Unit (TPU) boundaries across Hong Kong census years.

**Note:** Limited data available for analysis. Only 2006 TPU boundaries were successfully downloaded.

## Data Availability

- ✅ 2006: Available
- ❌ 2001: Not available
- ❌ 2011: Not available
- ❌ 2016: Not available
- ❌ 2021: Not available

## Recommendations

To complete the analysis, please:
1. Manually download TPU boundary data from the Esri China Open Data portal
2. Place the GeoJSON files in the `tpu_data/` directory
3. Re-run the analysis script
